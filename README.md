
## Installation

```
git clone https://github.com/WDYitz/shopping-cart.git
```

## Then 

```bash
npm install
```

## after that run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

### Abra [http://localhost:3000](http://localhost:3000) em seu navegador.
